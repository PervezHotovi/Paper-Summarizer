# 📄 AI PDF Summarizer

An AI-powered PDF Summarizer built using **LangChain**, **HuggingFace**, and **Streamlit**. This application allows users to upload any PDF document and automatically generates a concise and easy-to-understand summary using Large Language Models (LLMs).

---

## 🚀 Features

- 📂 Upload any PDF file
- 🧠 Automatic text extraction using PyPDFLoader
- ✂️ Smart text splitting for large documents
- 🤖 AI-generated summaries using HuggingFace LLMs
- 🌐 Simple and interactive UI with Streamlit

---

## 🛠️ Tech Stack

- LangChain
- HuggingFace
- Streamlit
- PyPDF
- Python
